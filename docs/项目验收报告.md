## 🚀 部署指南

### 前端部署（推荐Vercel）

```bash
# 1. 安装Vercel CLI
npm i -g vercel

# 2. 登录Vercel
vercel login

# 3. 部署
cd frontend
vercel
```

### 后端部署（推荐Docker）

```bash
# 1. 构建镜像
cd backend
docker build -t tech-learning-backend .

# 2. 运行容器
docker run -p 3001:3001 --env-file .env tech-learning-backend
```
