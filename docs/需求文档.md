# Tech Learning 技术资源网站需求文档

## 项目概述

**项目名称**: Tech Learning
**项目类型**: 技术资源综合平台
**版本**: v1.0
**创建日期**: 2024-01-15

## 1. 项目背景

随着AI技术的快速发展，开发者需要：

1. 获取最新的技术文章和学习资料
2. 高质量的Prompt模板来提升AI辅助开发效率
3. 快速找到MCP服务器和Agent Skill等AI工具资源

本项目旨在构建一个整合上述功能的技术资源平台，为开发者提供一站式学习体验。

## 2. 项目目标

### 主要目标

- 构建现代化的技术资源学习平台
- 提供美观、易用的用户界面
- 整合文章、Prompt模板、导航资源三大核心功能
- 支持微信扫码登录，降低使用门槛

### 技术目标

- 采用玻璃态设计风格，提供独特的视觉体验
- 实现60fps流畅动画效果
- 首屏加载时间 < 3秒
- Lighthouse性能分数 > 90

## 3. 功能需求

### 3.1 核心功能模块

#### 3.1.1 技术文章模块

**功能描述**: 展示和管理技术文章

**详细需求**:

- 文章列表展示
  - 支持分页加载
  - 支持按标签筛选
  - 支持关键词搜索
  - 显示文章标题、摘要、标签、阅读量、发布时间

- 文章详情页
  - Markdown内容渲染
  - 侧边目录导航
  - 相关文章推荐
  - 阅读量统计

- 文章管理（后台）
  - 文章创建、编辑、删除
  - 标签管理
  - 封面图片上传

#### 3.1.2 Prompt模板库模块

**功能描述**: 提供AI提示词模板，按用途分类

**详细需求**:

- 分类展示
  - 按用途分类（写作辅助、代码开发、数据分析、创意生成等）
  - 显示每个分类的模板数量

- 模板列表
  - 显示模板标题、描述、点赞数、复制次数
  - 支持按分类筛选
  - 卡片式布局，支持动画效果

- 模板详情页
  - 完整的Prompt内容展示
  - 一键复制功能
  - 点赞功能
  - 相关模板推荐

- 模板管理（后台）
  - 模板创建、编辑、删除
  - 分类管理
  - 统计数据（点赞数、复制次数）

#### 3.1.3 资源导航模块

**功能描述**: MCP服务器和Agent Skill资源导航

**详细需求**:

- 分类导航
  - MCP服务器
  - Agent Skills
  - 开发工具
  - 学习资源

- 资源展示
  - 卡片式布局
  - 显示资源标题、描述、访问次数
  - 外链跳转
  - 点击量统计

- 资源管理（后台）
  - 资源添加、编辑、删除
  - 分类管理
  - 访问量统计

### 3.2 用户功能

#### 3.2.1 认证与授权

- 微信扫码登录
- JWT Token认证
- 用户信息管理

#### 3.2.2 用户个人中心

- 个人资料编辑
- 收藏夹功能
- 浏览历史

### 3.3 搜索功能

- 全局搜索
- 按模块搜索
- 搜索建议
- 搜索历史

## 4. 非功能需求

### 4.1 性能需求

- 页面加载时间 < 3秒
- 动画帧率 ≥ 60fps
- 支持至少1000并发用户
- API响应时间 < 500ms

### 4.2 安全需求

- HTTPS加密传输
- SQL注入防护
- XSS攻击防护
- CSRF防护
- API访问频率限制

### 4.3 可用性需求

- 系统可用性 ≥ 99.5%
- 数据备份周期：每日
- 支持主流浏览器（Chrome、Firefox、Safari、Edge）

### 4.4 兼容性需求

- 响应式设计，支持PC、平板、手机
- 支持iOS 13+、Android 8+
- 支持Chrome 90+、Safari 14+、Firefox 88+、Edge 90+

## 5. 设计需求

### 5.1 视觉设计

#### 设计风格

- **玻璃态设计** (Glassmorphism)
  - 半透明背景
  - 背景模糊效果（backdrop-filter）
  - 细腻的边框和阴影
  - 渐变叠加层

- **主色调**: 绿色/青色系
  - 主色: #10B981 (Emerald 500)
  - 辅助色: #06B6D4 (Cyan 500)
  - 深色背景: #0F172A (Slate 900)

#### 动画特效

1. **微交互**
   - 按钮悬停：轻微放大 + 阴影增强
   - 卡片悬停：上浮 + 光泽效果
   - 输入框聚焦：边框发光动画

2. **页面转场**
   - 路由切换：淡入淡出 + 滑动效果
   - 页面加载：骨架屏 + 渐显动画

3. **滚动触发动画**
   - 元素视差滚动
   - 滚动时元素渐入
   - 卡片交错显现

### 5.2 用户体验设计

- 清晰的信息层级
- 直观的导航结构
- 流畅的操作反馈
- 一致的交互模式

## 6. 技术架构

### 6.1 技术栈

#### 前端

- **框架**: Next.js 15 (App Router)
- **语言**: TypeScript
- **样式**: TailwindCSS
- **动画**: Framer Motion
- **图标**: React Icons
- **HTTP客户端**: Axios

#### 后端

- **框架**: NestJS 10
- **语言**: TypeScript
- **ORM**: Prisma
- **数据库**: PostgreSQL
- **认证**: JWT
- **文档**: Swagger (可选)

### 6.2 系统架构

```
┌─────────────────────────────────────────────────────┐
│                     用户界面                         │
│              (Next.js + TailwindCSS)                │
└─────────────────────┬───────────────────────────────┘
                      │ HTTP/REST API
                      │
┌─────────────────────▼───────────────────────────────┐
│                   后端服务                           │
│              (NestJS + Prisma)                      │
├─────────────────────────────────────────────────────┤
│  模块：                                              │
│  - 认证模块 (Auth)                                   │
│  - 文章模块 (Articles)                               │
│  - Prompt模块 (Prompts)                              │
│  - 导航模块 (Navigation)                             │
└─────────────────────┬───────────────────────────────┘
                      │
┌─────────────────────▼───────────────────────────────┐

```

## 8. API设计

### 8.1 认证相关

- `GET /auth/wechat/qrcode` - 获取微信登录二维码
- `POST /auth/wechat/callback` - 微信登录回调
- `GET /auth/profile` - 获取用户信息

### 8.2 文章相关

- `GET /articles` - 获取文章列表
- `GET /articles/:id` - 获取文章详情
- `POST /articles` - 创建文章（需认证）
- `PUT /articles/:id` - 更新文章（需认证）
- `DELETE /articles/:id` - 删除文章（需认证）

### 8.3 Prompt相关

- `GET /prompts/categories` - 获取分类列表
- `GET /prompts` - 获取Prompt列表
- `GET /prompts/:id` - 获取Prompt详情
- `POST /prompts/:id/copy` - 增加复制次数
- `POST /prompts/:id/like` - 点赞

### 8.4 导航相关

- `GET /navigation/categories` - 获取分类列表
- `GET /navigation` - 获取导航链接
- `POST /navigation/:id/click` - 增加点击量

### 12.3 性能风险

- **风险**: 动画效果可能影响性能
- **应对**: 性能监控，必要时优化或禁用动画
